# Running a Golang Chat Application

## 1.  Installation  :

 ### Requirements :
```
Go - 1.13+
```
**Step-01 :** Download the project
```
https://github.com/kushtej/golang-chat-app.git
```
**Step-02 :** Setting up 
```
$ cd ./src
$ go get github.com/gorilla/websocket
$ go run main.go
```
**Step-03 :** Run the Browser : http://localhost:8080
